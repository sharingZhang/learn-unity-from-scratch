# 第1章：Unity入门介绍

## 1.1 什么是Unity？

Unity是一款强大的跨平台游戏引擎和开发工具，由Unity Technologies开发。它被广泛用于开发2D和3D游戏、交互式应用程序、虚拟现实（VR）和增强现实（AR）体验。

### Unity的主要特点

- **跨平台支持**：可以发布到Windows、Mac、Linux、iOS、Android、WebGL、游戏主机等多个平台
- **可视化编辑器**：直观的图形界面，无需编写代码即可创建游戏场景
- **强大的资源系统**：支持多种资源格式，包括模型、纹理、音频、动画等
- **丰富的组件系统**：通过组件化的方式快速构建游戏对象
- **活跃的社区**：拥有庞大的开发者社区和资源商店

## 1.2 Unity的应用领域

Unity不仅用于游戏开发，还广泛应用于：

- **游戏开发**：独立游戏、移动游戏、AAA游戏
- **教育培训**：交互式学习应用、模拟训练
- **建筑可视化**：建筑漫游、室内设计展示
- **影视制作**：实时渲染、虚拟制片
- **工业应用**：产品可视化、虚拟原型

## 1.3 安装Unity

### 步骤1：下载Unity Hub

1. 访问 [Unity官网](https://unity.com/)
2. 下载并安装Unity Hub
3. Unity Hub是管理Unity版本和项目的中心工具

### 步骤2：安装Unity Editor

1. 打开Unity Hub
2. 点击"安装"标签页
3. 点击"安装编辑器"
4. 选择Unity版本（推荐LTS版本，如2021.3 LTS）
5. 选择需要的模块：
   - **Microsoft Visual Studio Community**：代码编辑器
   - **Android Build Support**：Android开发（可选）
   - **iOS Build Support**：iOS开发（需要Mac，可选）
   - **WebGL Build Support**：Web开发（可选）

### 步骤3：创建Unity账号

- 在Unity Hub中登录或创建账号
- 个人开发者可以免费使用Unity Personal版本

## 1.4 创建第一个项目

### 创建新项目

1. 打开Unity Hub
2. 点击"新建项目"
3. 选择项目模板：
   - **2D**：2D游戏项目
   - **3D**：3D游戏项目
   - **3D (URP)**：使用通用渲染管线的3D项目
   - **3D (HDRP)**：使用高清渲染管线的3D项目
   - **VR**：虚拟现实项目
4. 设置项目名称和保存位置
5. 点击"创建项目"

### 项目结构

创建项目后，你会看到以下文件夹结构：

```
YourProject/
├── Assets/          # 游戏资源文件夹
│   ├── Scenes/      # 场景文件
│   ├── Scripts/     # 脚本文件
│   ├── Materials/   # 材质文件
│   ├── Prefabs/     # 预制体文件
│   └── ...
├── Packages/        # Unity包管理
├── ProjectSettings/ # 项目设置
└── UserSettings/    # 用户设置
```

## 1.5 Unity Editor界面概览

首次打开Unity Editor，你会看到以下主要窗口：

### 主要窗口

1. **Scene视图**：编辑和查看场景的3D/2D视图
2. **Game视图**：预览游戏运行效果
3. **Hierarchy窗口**：场景中所有游戏对象的层级结构
4. **Inspector窗口**：显示选中对象的详细属性和组件
5. **Project窗口**：项目资源文件浏览器
6. **Console窗口**：显示日志、警告和错误信息

### 窗口布局

Unity提供了多种预设布局：
- **Default**：默认布局
- **Tall**：高布局
- **Wide**：宽布局
- **2 by 3**：2x3布局

可以通过菜单 `Window > Layouts` 切换布局。

## 1.6 基本操作

### 场景视图操作

- **旋转视角**：按住鼠标右键并拖动
- **平移视角**：按住鼠标中键并拖动（或按住Alt+鼠标左键）
- **缩放视角**：滚动鼠标滚轮
- **聚焦对象**：选中对象后按F键

### 创建游戏对象

1. 在Hierarchy窗口中右键点击
2. 选择 `Create Empty` 创建空对象
3. 或选择其他预设对象（如Cube、Sphere、Plane等）

### 移动、旋转、缩放对象

使用工具栏的变换工具：

- **移动工具**（W键）：移动对象位置
- **旋转工具**（E键）：旋转对象
- **缩放工具**（R键）：缩放对象大小
- **矩形工具**（T键）：2D对象的矩形变换

## 1.7 保存和打开项目

### 保存场景

- 按 `Ctrl+S`（Windows）或 `Cmd+S`（Mac）
- 或通过菜单 `File > Save`

### 打开项目

- 在Unity Hub中点击"打开"按钮
- 选择项目文件夹

## 1.8 运行游戏

### 播放模式

1. 点击Scene视图或Game视图上方的播放按钮（▶）
2. 游戏开始运行
3. 再次点击停止播放

### 注意事项

- 播放模式下对场景的修改不会保存
- 退出播放模式后，所有修改会恢复
- 这是Unity的安全机制，防止误操作
